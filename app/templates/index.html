Development Process

Get the Data & Process the DaTA

We found the data on Kaggle, then, we used pandas, regex, and nltk to clean the data and pre-process it into a format we could use to train the model.

Implement the Model

With word2vec and SciKitLearn, we got 75% accuracy with GloVe and Tensorflow Keras, we got 82% accuracy.

Develop the Website

We utilized Nicepage to create the website. Changes were made on a daily basis to accommodate what each member wanted to present.

Deployment

We use flask to utilize the prompt we get from html, then run the model and generate the predictions showing up on the website.
